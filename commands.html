Bash Commands Encyclopedia

This page serves as a comprehensive encyclopedia for Bash commands. Whether you are a beginner or an advanced user, explore the detailed explanations, examples, and usage tips for various commands that power your Unix/Linux systems.

File and Directory Management Commands

ls

Description: Lists directory contents.

Usage: ls [options] [directory]

Example: ls -l /home/user shows a detailed list of files.

cd

Description: Changes the current working directory.

Usage: cd [directory]

Example: cd /var/log navigates to the log directory.

pwd

Description: Prints the current working directory.

Usage: pwd

Example: Simply type pwd to see your current path.

mkdir

Description: Creates a new directory.

Usage: mkdir [options] directory

Example: mkdir new_folder creates a directory named "new_folder".

rmdir

Description: Removes an empty directory.

Usage: rmdir directory

Example: rmdir old_folder deletes an empty folder.

cp

Description: Copies files and directories.

Usage: cp [options] source destination

Example: cp file.txt backup_file.txt copies a file.

mv

Description: Moves or renames files and directories.

Usage: mv [options] source destination

Example: mv oldname.txt newname.txt renames a file.

rm

Description: Removes files or directories.

Usage: rm [options] file

Example: rm -r directory removes a directory and its contents.

find

Description: Searches for files and directories based on conditions.

Usage: find [path] [expression]

Example: find /home/user -name "*.txt" finds all text files.

File Content Viewing and Editing

cat

Description: Concatenates and displays file contents.

Usage: cat [options] file

Example: cat /etc/passwd displays the content of the passwd file.

less

Description: Views file contents one page at a time.

Usage: less file

Example: less /var/log/syslog for paging through log files.

more

Description: Another pager to view file contents sequentially.

Usage: more file

Example: more README.md shows the README file page by page.

head

Description: Displays the first few lines of a file.

Usage: head [options] file

Example: head -n 10 file.txt shows the first 10 lines.

tail

Description: Displays the last few lines of a file.

Usage: tail [options] file

Example: tail -f logfile.log continuously displays the end of a log file.

Process Management Commands

ps

Description: Displays information about active processes.

Usage: ps [options]

Example: ps aux shows detailed process information.

top

Description: Provides a real-time view of running processes.

Usage: Simply type top

Example: Run top to monitor system performance and processes.

kill

Description: Sends a signal to terminate a process.

Usage: kill [options] PID

Example: kill -9 1234 forcefully terminates process with PID 1234.

pkill

Description: Terminates processes based on name or attributes.

Usage: pkill [options] pattern

Example: pkill firefox terminates all Firefox processes.

nice & renice

Description: Adjusts the priority of running processes.

Usage: Use nice when launching a process, and renice to change the priority of an existing process.

Example: nice -n 10 command starts a command with lower priority.

File Permissions and Ownership

chmod

Description: Changes the file permissions.

Usage: chmod [options] mode file

Example: chmod 755 script.sh sets the file to be executable.

chown

Description: Changes the file owner and group.

Usage: chown [options] owner:group file

Example: chown user:staff document.txt changes ownership.

chgrp

Description: Changes the group ownership of a file or directory.

Usage: chgrp [options] group file

Example: chgrp developers project_folder assigns a new group.

Networking Commands

ping

Description: Checks the network connectivity to a host.

Usage: ping [options] host

Example: ping google.com to test connectivity.

ifconfig / ip

Description: Configures or displays network interface parameters. ifconfig is traditional, while ip is the modern replacement.

Usage: ifconfig or ip addr show

Example: ip addr displays current network settings.

netstat

Description: Displays network connections, routing tables, and interface statistics.

Usage: netstat [options]

Example: netstat -tulpn shows active connections and listening ports.

ssh

Description: Securely connects to a remote machine via the SSH protocol.

Usage: ssh [options] user@host

Example: ssh user@example.com initiates a secure connection.

scp

Description: Securely copies files between hosts over SSH.

Usage: scp [options] source destination

Example: scp file.txt user@example.com:/path/to/destination transfers a file.

System Information Commands

uname

Description: Displays system information.

Usage: uname [options]

Example: uname -a prints all available system information.

df

Description: Shows disk space usage for file systems.

Usage: df [options]

Example: df -h presents human-readable output.

du

Description: Estimates file and directory space usage.

Usage: du [options] [file/directory]

Example: du -sh /var/log shows the size of the log directory.

free

Description: Displays memory usage statistics.

Usage: free [options]

Example: free -m shows memory usage in megabytes.

uptime

Description: Tells how long the system has been running along with load averages.

Usage: uptime

Example: Simply type uptime to see the system's runtime.

Text Processing Commands

grep

Description: Searches for patterns within files.

Usage: grep [options] pattern file

Example: grep "error" /var/log/syslog finds lines containing "error".

sed

Description: Stream editor for filtering and transforming text.

Usage: sed [options] 's/pattern/replacement/' file

Example: sed 's/old/new/g' file.txt replaces "old" with "new".

awk

Description: A powerful text processing language and command.

Usage: awk 'pattern {action}' file

Example: awk '{print $1}' file.txt prints the first field of each line.

sort

Description: Sorts lines of text files.

Usage: sort [options] file

Example: sort file.txt sorts the file alphabetically.

uniq

Description: Filters out repeated lines in a file.

Usage: uniq [options] file

Example: sort file.txt | uniq prints unique lines from a sorted file.

cut

Description: Removes sections from each line of files.

Usage: cut [options] file

Example: cut -d: -f1 /etc/passwd extracts the first field from the passwd file.

tr

Description: Translates or deletes characters.

Usage: tr [options] set1 set2

Example: echo "hello" | tr 'a-z' 'A-Z' converts text to uppercase.

wc

Description: Counts lines, words, and characters in text.

Usage: wc [options] file

Example: wc -l file.txt counts the lines in a file.

Shell Built-in and Utility Commands

echo

Description: Displays a line of text/string passed as an argument.

Usage: echo [options] [string]

Example: echo "Hello, Bash!" prints text to the terminal.

history

Description: Displays the command history.

Usage: history

Example: Type history to view previously executed commands.

alias

Description: Creates shortcuts for commands.

Usage: alias name='command'

Example: alias ll='ls -l' sets a shortcut for a long listing.

export

Description: Sets environment variables for child processes.

Usage: export VARIABLE=value

Example: export PATH=$PATH:/new/path appends a new path.

source (.)

Description: Reads and executes commands from a file in the current shell environment.

Usage: source filename or . filename

Example: source ~/.bashrc reloads your Bash configuration.

Advanced Commands and Scripting Utilities

xargs

Description: Builds and executes command lines from standard input.

Usage: command | xargs [options] command

Example: find . -name "*.txt" | xargs grep "search_term" searches for a term within found text files.

tar

Description: Archives multiple files into a single file, with optional compression.

Usage: tar [options] archive_name.tar file(s)

Example: tar -czvf archive.tar.gz folder/ creates a compressed archive.

zip / unzip

Description: Compresses (zip) or extracts (unzip) files.

Usage: zip archive.zip file(s) and unzip archive.zip

Example: Compress files using zip backup.zip *.txt, then extract with unzip backup.zip.

curl / wget

Description: Retrieves data from URLs. curl is versatile for API requests, while wget is simple for downloading files.

Usage: curl [options] URL or wget [options] URL

Example: curl -O http://example.com/file.zip downloads a file.

printf

Description: Formats and prints data. More powerful than echo for formatted output.

Usage: printf format [arguments]

Example: printf "Name: %s\n" "Bash" prints a formatted string.

Miscellaneous and Utility Commands

date

Description: Displays or sets the system date and time.

Usage: date [options]

Example: date "+%Y-%m-%d %H:%M:%S" shows the current date and time.

sleep

Description: Pauses execution for a specified period.

Usage: sleep seconds

Example: sleep 5 pauses for 5 seconds.

time

Description: Measures the execution time of a command.

Usage: time command

Example: time ls displays how long the ls command took to execute.

man

Description: Displays the manual pages for other commands.

Usage: man command

Example: man grep shows the manual for the grep command.
